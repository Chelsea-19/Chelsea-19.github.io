Bio · Foundation Models · NTU Research

Evo 2 Genomic Language Model

围绕 Acinetobacter baumannii 设计了多尺度解释框架，覆盖 replicon-level、fixed-window、dynamic-window 与 embedding 分析，用于研究物种层面的 genomic regularity 与异常区域。

  • 搭建了长序列推理、gene-level perplexity 与表示提取分析流程。
  • 把探索性分析延伸到更大的 genomic agent、retrieval 与 distillation 方向。
  • 把工作定位为可解释 AI4Science 研究，而不是单纯 benchmark。
意义：说明 foundation model 信号不仅可用于预测，也可用于生物学解释。

Healthcare · Clinical AI · Parexel

Patient Recruitment Agent (PRA)

参与 AI 辅助临床患者招募工作流，重点关注 eligibility logic、evidence extraction 与下游存储设计如何适配真实监管和业务约束。

  • 将临床文本映射为结构化筛选条件与 workflow-ready 表示。
  • 分析 agent 设计在 APAC CRO 与 RWD/RWE 场景下的适配方式。
  • 关注 workflow quality，而不仅仅是 extraction accuracy。
意义：体现医疗 AI 经验是贴近落地现实的，而不是 toy demo。

Healthcare · Medical Education · Agent Systems

SimuPatient

将一个 LLM 标准化病人原型进一步重构成更适合研究的教育型 agent system，加强了 reproducibility、persona consistency 与结构化评估。

  • 从 case generation 到 consultation、logging、scoring 重新整理系统流程。
  • 引入 hidden-case logic 与更严格的评估设计。
  • 提升了从 prototype 到 research artifact 的路径清晰度。
意义：体现医疗教育 AI 中的评估意识与研究方法意识。

Public Health · Evidence Agent · Drug Repurposing

OrphanCure

构建了一个可验证的生物医学 agent 原型，用于罕见病药物重定位，串联 entity resolution、target retrieval、文献检索、分阶段推理与 post-hoc evidence checking。

  • 将 agent 设计成带有明确状态和 human-in-the-loop 控制的流程。
  • 集成 OpenTargets、PubMed 等检索工具。
  • 通过 schema-constrained outputs 与 verification 降低 hallucination 风险。
意义：体现公共卫生与转化医学推理中的可审计证据链。
